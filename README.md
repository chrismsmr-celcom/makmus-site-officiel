## Présentation

MAKMUS est un site d'actualités interactif qui couvre plusieurs catégories : Politique, Économie, Sport, Musique, RDC et Jeux. Chaque section propose un contenu riche et actualisé via une API, avec un design inspiré des grands médias internationaux.  

La catégorie **Jeux** offre des mini-jeux intégrés directement dans le site pour divertir les visiteurs : Sudoku, Serpent, Mots croisés et Échecs.

---

## Fonctionnalités

- Interface responsive, adaptée aux mobiles, tablettes et desktops.
- Menu fullscreen moderne avec barre de recherche interne.
- Barre de date et heure dynamique.
- Sections d’actualités par catégorie avec chargement via API.
- Mini-jeux interactifs dans la section Jeux.
- Footer complet avec liens vers toutes les pages légales et informatives.
- Accordéons pour des sections supplémentaires comme les contacts.
- Styles premium inspirés des grands médias (NY Times).

---

## Technologies utilisées

- HTML5, CSS3, JavaScript ES6
- Google Fonts : Libre Baskerville, Inter
- API externe pour les articles d’actualités (NewsData.io)
- Canvas HTML pour Snake et Échecs
- Flexbox et Grid pour la mise en page responsive
